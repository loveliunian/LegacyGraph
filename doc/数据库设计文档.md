# LegacyGraph 数据库设计文档

## 📋 概述

LegacyGraph 使用 **PostgreSQL 18+** 作为主数据库，**Neo4j 5.x** 作为图数据库存储图谱关系，**pgvector** 扩展用于向量相似性检索。

本文档描述 PostgreSQL 数据库的表结构设计。

---

|------|------|------|------|

| `project_code` | VARCHAR(128) | UNIQUE NOT NULL | 项目编码 |

| `description` | TEXT | - | 项目描述 |

| `default_branch` | VARCHAR(128) | - | 默认分支 |

| `created_at` | TIMESTAMP | NOT NULL DEFAULT CURRENT_TIMESTAMP | 创建时间 |

## 🔌 扩展说明

### pgvector 扩展
项目使用 `pgvector` 扩展存储文档嵌入向量，需要在数据库创建扩展：
```sql
CREATE EXTENSION IF NOT EXISTS vector;
```

### JSONB 使用
所有灵活属性都使用 JSONB 类型存储，支持：
- 任意结构的动态数据
- GIN 索引支持快速查询
- PostgreSQL 原生 JSON 操作符
