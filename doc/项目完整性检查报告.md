## 项目概述

**LegacyGraph** 是一个基于知识图谱和 LLM 的遗留系统迁移分析平台。

- **架构：** Spring Boot + Vue 3 前后端分离
- **存储：** PostgreSQL + pgvector + Neo4j
- **AI 集成：** Spring AI + OpenAI
- **核心能力：** 自动化代码理解 → 事实抽取 → 图谱构建 → 测试生成 → 迁移评估

## 快速开始

### 环境要求

| 软件 | 最低版本 |
| :------ | ---------: |
| Docker | 20.x |
| Docker Compose | 2.x |

### 一键部署

```bash
# 克隆项目
git clone <repository-url>
cd LegacyGraph/deploy

# 启动所有服务
docker-compose up --build -d

# 查看日志
docker-compose logs -f
```
