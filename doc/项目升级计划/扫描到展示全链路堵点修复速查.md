## 风险与决策点

| 风险 | 触发条件 | 缓解 |
|------|---------|------|
| SourceConnector 切流后性能退化 | 切流 1 周内 ADAPTER_SCAN p95 增长 >30% | 保留 Feature Flag 一键回滚 |
| `/graph/unified` 流式后前端兼容 | 旧客户端调用未带 cursor 报错 | 服务端缺 cursor 时返回首屏 + 旧形态 |
| L-09 注解优先后旧节点失配 | 旧版本扫描结果突然空 | 旧节点 label 不动，新扫描自然覆盖 |
| `versionsCache` 主动失效风暴 | 多 tab 同时刷新 | 1s 节流合并 clearVersionsCache |
| `@ts-expect-error` 修复后 Vue Flow 行为不一致 | staging 居中/缩放异常 | 锁版本号，2 周内仅 patch 不升级 |
