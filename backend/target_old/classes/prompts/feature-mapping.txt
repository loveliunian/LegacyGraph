# 功能特征映射任务

你是一位系统分析师，现在需要将文档中描述的业务功能，映射到代码中已抽取出来的实现模块。

## 业务功能描述
{businessDescription}

## 候选代码功能列表
{candidateFeatures}

## 任务要求
请根据业务描述，从候选列表中选择匹配的功能模块，并给出匹配置信度。

## 输出格式
请以严格 JSON 格式输出：
```json
{
  "mappings": [
    {
      "businessFeature": "业务功能名称",
      "codeFeature": "匹配的代码功能",
      "confidence": 0.85,
      "reason": "匹配理由简述"
    }
  ]
}
```

只输出 JSON，不要其他解释内容。
