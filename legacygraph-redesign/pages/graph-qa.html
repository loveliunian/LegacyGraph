订单服务和支付服务之间有哪些关联?
图谱助手
订单服务和支付服务之间存在以下关联：
  1. 订单创建时调用支付接口完成支付
  2. 支付回调更新订单状态
  3. 共享用户认证模块 (AuthModule)
  4. 通过 MySQL 数据库共享订单表和支付记录表
你
哪些服务依赖数据库?
图谱助手
以下服务依赖数据库：
  • 订单服务 — MySQL (主库 + 从库)
  • 用户服务 — MySQL (用户信息库)
  • 库存服务 — MySQL + Redis (热点缓存)
  • 支付服务 — MySQL (支付流水库)
